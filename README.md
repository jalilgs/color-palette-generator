# Color Palette Generator

A simple web app that generates random color palettes, with the ability 
to lock colors, copy hex codes, and save favorite palettes.

## Features
- Generate random 5-color palettes
- Lock individual colors so they stay fixed on regenerate
- Copy hex codes to clipboard with one click
- Save palettes for later, with duplicate detection
- Responsive design

## Built with
- HTML
- CSS
- JavaScript (vanilla, no frameworks)
